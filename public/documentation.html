Movie API

This API provides information about movies, directors, and genres. It is designed to support a client-side application that lets users browse movies, view details, and manage their favorite films.

API Endpoints

Request URL Method Description Query Parameters Request Body Response
Get all movies /movies GET Returns a list (array) of all movies in JSON format None None JSON array of movie objects
Get single movie by title /movies/:title GET Returns JSON with details for that movie (description, genre, director, image URL, featured) title - movie title in URL path None JSON object with movie details
Get genre info by name /genres/:name GET Returns JSON with the genre’s name and description name - genre name in URL path None JSON object with genre details
Get director info by name /directors/:name GET Returns JSON with director’s bio, birth year, death year name - director name in URL path None JSON object with director details
Register a new user /users POST Creates a new user account None { "username": "sara", "email": "sara@example.com", "password": "password123" } Confirmation message and JSON with created user info
Update a user’s info /users/:username PUT Updates user’s username username - current username in URL path { "username": "newusername" } Confirmation message that user info was updated
Add a movie to favorites /users/:username/favorites/:movieId POST Adds a movie to user's favorites list username - user’s username
movieId - movie's ID in URL path		 None Text message confirming movie was added
